App数据抓包必需工具
必需工具：小米手机，Charles，HttpCanary

从2023年2月后，京东只限于从app发起抢购，所以，网上的很多工具已经无效了。只能分析app端的底层协议和流程。

通过抓包可以发现，整个抢购流程分为七个步骤，如下：
